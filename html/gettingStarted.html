Swordfish 入门指南

只需几个简单步骤即可使用 Swordfish 进行翻译。基本的文档翻译流程如下：

  1. 设置记忆库和术语库
  2. 创建项目
  3. 翻译项目
  4. 导出译文

提示：

随时按 F1 打开 Swordfish 用户手册 获取更多信息。

步骤 1：设置记忆库和术语库

Swordfish 使用 翻译记忆库（TM）技术辅助译员。它会将你的译文存储在 记忆库 中，并在你需要翻译相似文本时再次提供建议。

  1. 点击 添加记忆库 按钮（）在 记忆库 标签栏工具栏上，为你的译文创建一个记忆库。
  2. 如果你有 TMX 格式的翻译记忆数据，可点击 导入 TMX 文件 按钮（）导入 TMX 文件到你的记忆库。

使用 术语库 存储常用术语及其译文，便于翻译时查阅。术语库中的词条还可辅助 Swordfish 内置的 自动翻译 引擎进行匹配。

  1. 点击 添加术语库 按钮（）在 术语库 标签栏工具栏上，按提示填写相关信息。
  2. 如果你有 TMX 或 TBX 格式的术语数据，可点击 导入术语库 按钮（）导入术语数据到你的术语库。

提示：

你可以在不同翻译项目中复用记忆库和术语库，无需每次新建项目都新建记忆库或术语库。

步骤 2：创建项目

Swordfish 支持一次翻译一个或多个文件。翻译任务以 项目 形式组织。

创建项目有两种方式：

  • 点击 翻译单个文件 按钮（）在 项目 标签栏工具栏，使用简化对话框创建包含单个文件的项目。
  • 点击 添加项目 按钮（）在 项目 标签栏工具栏，创建包含一个或多个文件的项目。

从单个文件创建项目时，文件名和路径会作为项目名称。

步骤 3：翻译项目

新建项目会自动打开进行翻译。若要继续翻译已关闭的项目：

  1. 在 项目 标签页中，点击你要翻译的项目。
  2. 点击 翻译项目 按钮（）在 项目 标签栏工具栏。

翻译视图包含以下区域：

顶部工具栏 包含执行不同任务的按钮，以及用于选择当前记忆库和术语库的下拉菜单。
翻译面板 大型可滚动的片段表格，显示源文本、片段状态和译文。
侧边面板 位于右侧，包含 翻译记忆库、机器翻译 和 术语 三个面板。
导航与状态栏 位于底部，包含项目页面导航按钮及翻译状态信息。

点击某个片段即可开始翻译。对应行会高亮并尽量居中显示，光标自动定位到目标文本开头。

源文格式通过带数字的绿色标记（如：）表示。插入绿色标记（即"标签"）时，Windows/Linux 下按 Ctrl+N，macOS 下按 Cmd+N，其中 N 为标记上的数字。插入大于 10 的标签时，按 Ctrl+T（macOS 下为 Cmd+T），并使用弹窗输入。你也可以在 编辑 菜单中使用标签管理相关功能。

提示：

在片段被确认后，翻译面板 中心列会出现一个警告图标（）。

当片段完全翻译后，按 Ctrl+E (Cmd+E on macOS) 或点击顶部工具栏上的 确认翻译 按钮（）确认。确认片段后，其翻译会被存储在顶部工具栏中选择的记忆库中，并自动传播到所有未确认的相似片段。

你有两个选项可以将片段标记为确认：

  • 按 Alt+Down Arrow (Ctrl+Alt+Down Arrow on macOS) 或点击顶部工具栏上的 确认并转到下一个未翻译 按钮（）确认当前片段并打开下一个未翻译的片段。
  • 按 Alt+Shift+Down Arrow (Ctrl+Shift+Down Arrow on macOS) 或点击顶部工具栏上的 确认并转到下一个未确认 按钮（）确认当前片段并打开下一个未确认的片段。

使用 Page Up 和 Page Down 键移动到 下一个和上一个片段并翻译它们。

当所有片段都被翻译后，按 F5 或点击顶部工具栏上的 导出 HTML 按钮，导出翻译项目的 HTML 版本以供审查。注意任何带有错误的片段并进行修正。

要查看知道片段编号，按 Ctrl+G (Cmd+G on macOS)。转到片段 对话框出现。在 片段编号 框中输入要查看的片段编号并按 Enter。

按 F9 或点击顶部工具栏上的 检查内联标签 按钮（）检查对话框中出现的项目，确保没有标签错误，因为翻译后的文档可能无法使用。

要验证所有初始和尾随空格是否正常，按 F10 或点击顶部工具栏上的 检查初始/尾随空格 按钮（）。此任务对于某些亚洲目标语言不是必需的。

步骤 4：导出译文

一旦所有片段都被翻译和审查，生成翻译项目的翻译版本。

点击顶部工具栏上的 导出翻译 按钮（）或 项目 标签栏工具栏上的 导出翻译 按钮。操作系统将显示一个对话框，用于选择保存翻译文档的位置。

翻译完成后，将出现一个新对话框，询问是否打开翻译后的文件或文件夹。

重要提示

  • 浏览应用程序菜单，了解所有可用选项。
  • 检查 Swordfish 用户手册 获取更多信息（随时按 F1）。
  • 加入 Maxprograms 支持小组 并询问你可能需要的任何帮助。
